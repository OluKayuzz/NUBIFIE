# NUBIFIE Web Portal

## Overview

The NUBIFIE Web Portal is a dynamic web application built using ASP.NET Core MVC for the National Union of Banks, Insurance and Financial Institutions Employees (NUBIFIE). The platform provides a centralized system for communication, news publishing, member engagement, and organizational information management.

## Key Features

* News and announcements management with categorization
* Event publishing and updates
* Member-focused information and communication
* Dynamic content management system (CMS-style structure)
* Administrative dashboard for managing content
* Responsive UI built with Bootstrap 5

## Technology Stack

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* Bootstrap 5
* JavaScript / jQuery
* HTML5 & CSS3

## Architecture

The system follows the MVC (Model-View-Controller) pattern to ensure clean separation of concerns:

* **Models** handle data structure and business logic
* **Views** manage UI rendering
* **Controllers** process user requests and application flow

## Setup Instructions

1. Clone the repository
2. Configure the database connection string in `appsettings.json`
3. Run database migrations (if applicable)
4. Build and run the project using Visual Studio or `dotnet run`
5. Access the application via `https://localhost:<port>`

## Project Status

This project is actively developed and maintained, with continuous improvements focused on usability, performance, and content management capabilities.

## Purpose

This system was built to modernize communication and information sharing within the NUBIFIE organization by providing a reliable, scalable, and easy-to-manage digital platform.
